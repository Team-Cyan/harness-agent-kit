# Delivery Checklist

Before calling a task done, check:

- [ ] Scope stayed inside one module or one roadmap item
- [ ] Focused verification commands were run
- [ ] Output does not expose secrets
- [ ] Durable knowledge went into `docs/`, not `.agents/`
- [ ] Relevant docs were updated if project state changed
- [ ] `docs/roadmap.md` reflects the new status
- [ ] `docs/operations/session-handoff.md` was updated if future sessions need the nuance
