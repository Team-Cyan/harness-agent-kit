# Repo Init Kit Module

## Purpose

Own the reusable docs, templates, and path conventions that help bootstrap agent-first repositories.

## Primary Files

- `docs/ai/repo-init-kit/README.md`
- `docs/ai/repo-init-kit/path-architecture.md`
- `docs/ai/repo-init-kit/bootstrap-checklist.md`
- `docs/ai/repo-init-kit/templates/`

## Current Responsibilities

- define the recommended repository layout,
- provide reusable templates,
- document adoption modes and profiles,
- keep the kit practical and copyable.

## Verification

- manually inspect links and paths
- ensure templates stay internally consistent
